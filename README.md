## Regular Expression

* Ký số: \d hoặc [0-9]
* [xyz]: hoặc x, hoặc y, hoặc y (lấy 1 trong nhóm)
* [^xyz]: ko bắt đầu bởi x
* (A|B|C)

### Bản số:
* *: lặp 0 .. n lần
* +: lặp 1 .. n lần (tối thiểu 1 lần)
* {m}: chính xác m lần
* {m, n}: từ m đến n lần

Khoanh nhóm dùng ()
